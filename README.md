# 北极地缘与技术多要素联动3D可视化平台

## 项目概述

「北极地缘与技术多要素联动3D可视化平台」是基于「大北极」格局下地缘与技术双向互动机制研究开发的数字化决策参考系统。

## 核心技术

- **框架**：Python 3.10+ / Streamlit
- **可视化**：Plotly 3D Globe / Folium / PyDeck
- **部署**：Streamlit Community Cloud
- **数据**：GDELT / NSIDC / Patent DB

## 安装运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 项目结构

```
beiji/
├── app.py                    # 主入口
├── pages/
│   ├── 2_气候时空监测.py    # 气候模块
│   ├── 3_地缘战略格局.py    # 地缘模块
│   ├── 4_极地核心技术.py    # 技术模块
│   ├── 5_中国安全风险.py    # 安全模块
│   ├── 6_数据中心工具.py    # 数据模块
│   └── 7_关于本项目.py      # 关于模块
├── src/
│   ├── data_core.py          # 数据层
│   └── viz.py                # 可视化库
├── data/processed/           # 处理后数据
└── geojson/                  # 地理数据
```

## 六大核心模块

| 模块 | 功能 |
|------|------|
| 气候监测 | 1980-2025海冰数据 + CMIP6情景预测 |
| 地缘格局 | 大北极博弈网络 + 政策文本NLP |
| 技术竞争 | 专利气泡图 + 技术-地缘联动 |
| 安全风险 | 四维风险矩阵 + SWOT + 策略推演 |
| 数据中心 | 数据下载/上传/查询/对比 |
| 关于项目 | 项目成果与架构说明 |

## 数据说明

平台数据均为基于公开资料模拟的示例数据，供功能演示使用。真实数据请替换 `data/processed/` 目录下的文件。
